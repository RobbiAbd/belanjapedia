## Catatan Teknis
- Integrasi logic konversi koin: 50 skor game = 1 koin (dibatasi maks 50 koin per game).
- Penyimpanan papan peringkat global dilakukan menggunakan file JSON lokal di `server/data/leaderboard.json`.
- Kecepatan Kucing dihitung sebagai: `2 * Math.pow(1.15, level - 1)`. Kecepatan Tikus dihitung sebagai: `playerSpeed * 0.75`.
- Jumlah spawn musuh per spawn event: `1 + Math.floor(timeElapsed / 5)`.
- Tembakan bertambah searah sebaran sudut radial (0.25 radian) berdasarkan tingkat level kucing.
- Seluruh unit test (Vitest) berhasil dijalankan secara lokal dengan hasil sukses 100% (86/86 passed, termasuk 14 unit test baru untuk game).
